Diving Into the Mind of a Programmer

Hello World was one of the first things ever programmed. Because of this, many programmers first learn how to output the sentence "Hello World!" to their screen when beginning to learn a new programming language. The exact method of doing so varies between languages, and an example of this can be found at the bottom of this page.


A few examples of the code used to print Hello World! to a user's screen are provided below, alongside the language that each snippet is written in.

Notes: The code snippets only show the actual code that executes the command to print out the phrase "Hello World!" to the user, not any code required to make that command work without errors. Ex. A Java "Hello World!" program uses a main method, which is not included on this page.
In the cases where there are multiple correct answers, the word or will be placed between the two ways of printing "Hello World!". The word or is NOT part of the code to print the phrase "Hello World!" to the screen of the user.

  • Coding Language: Python
    print("Hello World!")
  • Coding Language: Java
    System.out.println("Hello World!"); or System.out.print("Hello World!");
  • Coding Language: C#
    Console.WriteLine("Hello World!"); or Console.Write("Hello World!");
